# 🐝 이슈 트래커
- 2023 코드스쿼드 마스터즈 Max에서 진행한 "이슈 트래커"를 구현하는 그룹 프로젝트
- 이슈 트래커는 문제를 해결할 이슈를 작성하고 이슈에 추가할 수 있는 옵션인 레이블, 마일스톤 등을 추가하여 이슈를 편리하게 관리할 수 있는 애플리케이션입니다.
- 미션 기간: `2023-07-24 ~ 2023-08-18` (1달)
